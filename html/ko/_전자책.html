경희전자책 도서관
  • 전 주제 분야의 국내전자책 통합검색
  • 대출 책수 : 5책 5일(1 회 연장가능)
  • 전자책 대출·예약현황 : My Library > My 자료 > 전자책 대출·예약현황에서 확인 가능
  • 최초 이용 시 반드시 PC 또는 모바일웹을 통한 도서관 홈페이지 로그인 필요 [Mobile 이용 안내]
  • 모바일 이용
    • 앱 다운로드 및 경희대학교 도서관 접속
      • 교보도서관 앱 다운로드→도서관 검색→'경희대학교' 접속
      • 북큐브 앱 다운로드→도서관 검색→'경희대학교 전자책 도서관' 접속
    • 로그인 정보 : ID 학번 또는 사번 / PW 학번 또는 사번
    • 킨들, 크레마 등 전자책 전용 기기 지원안됨
교보 전자책 (구독형 서비스)
  • 교보문고 국내 전자책 중 구독형으로 제공하는 전자책
  • 대출 책수 : 10책 14일 (전자책 7권/오디오북 3권), 연장 불가
  • 최초 이용 시 반드시 PC 또는 모바일웹을 통한 도서관 홈페이지 로그인 필요 [이용안내]
    • 기존 경희전자책도서관과 다른 별도 로그인 필요
  • 모바일 이용
    • 교보도서관 앱 다운로드 → 도서관 검색 -> '경희대학교(구독)' 접속
    • 로그인 정보 : ID 학번 또는 사번 / PW 학번 또는 사번
  • 경희대 본교 구성원만 이용 가능 (산단 소속, 졸업·수료생 등 이용 불가)
APABI
  • 북경대, 청화대, 복단대, 절강대, 중국인민대 대학출판사 전자책
  • 동시접속자 수 : 3명
Cambridge University Press Ebook(EBA)
  • Cambridge Core 플랫폼 내 Access 가능한 전주제 분야 약 18,000 여 종 단행본 (일부 textbook, referencebook 제외)
  • 제공 연도: 2012년~2022년
  • 상세 매뉴얼
Ebook Central
  • 전 주제 분야의 해외 전자책
  • 개인계정 생성 권장
    • Ebook Central 개인계정 필요한 경우
      • 5분 무료 이용(온라인 읽기만 가능) 도서의 구매 요청 : 구매 요청 이후 챕터 다운로드 가능
      • 이용가능 도서의 전체 다운로드 또는 인쇄
    • Ebook Central 개인계정이 필요없는 경우
      • 이용가능 도서의 챕터 다운로드(5분 무료 이용 도서는 해당없음) 또는 온라인 읽기
  • 2021~2023 KERIS 대학라이선스 100% 지원: ProQuest eBook Reference Collection (참고도서 870여권 제공)
  • 상세 매뉴얼
